Implementation

Dead-Simple Implementation

Adding your AI chatbot to your website takes less than 5 minutes. No coding skills required.

index.html 
<!-- JAYISAAC AI Chatbot -->

<script src="https://cdn.botpress.cloud/webchat/v3.3/inject.js"></script>
<script src="https://files.bpcontent.cloud/YOUR-BOT-ID/webchat.js"></script>

<!-- That's it! Your AI chatbot is now live. -->

Any Website

Works with HTML, React, Vue, Angular. Just paste the code before your closing body tag.

WordPress

Use our plugin or add code via theme editor or header/footer plugins.

Shopify

Add to theme.liquid or use our Shopify app for one-click installation.

Wix

Use Wix's "Add Custom Code" feature in site settings.

Squarespace

Navigate to Settings → Advanced → Code Injection.

Custom Apps

Full API access for mobile apps, Slack, Teams, WhatsApp.

Common Questions

+
Absolutely not. We handle all the technical work. You just copy and paste a simple code snippet.

+

+
100% customizable. Colors, fonts, avatar, position, welcome message — everything can be tailored to your brand.
